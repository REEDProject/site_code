A validation error occurred at or following the marked point in {{ filename }}:

{{ error }}

Very often the error is one of improper nesting of codes (incorrect: "@i\@a\...@i \@a \"; correct: "@i\@a\...@a \@i \"), or the closing code not including the space before the backslash (incorrect: "@i\...@i\"; correct: "@i...@i \"). In such cases, the mark for where the error occurred points to the start of the range that contains the error.

Errors pointing to the opening of a table ("<t>") typically occur when there is a problem within that table's contents.
